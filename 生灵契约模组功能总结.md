## 🎯 核心概念

### 契约系统

生灵契约模组的核心玩法围绕着**契约**与**阵营对抗**展开：

| 概念 | 说明 |
|:---:|:---|
| 📜 **生灵契约** | 与特定模组的生物签订契约，该模组的生物将不再攻击你 |
| 👥 **队伍系统** | 组建队伍，共享契约效果，队友之间不会造成误伤 |
| ⚔️ **阵营对抗** | 不同契约阵营的生物会自动互相攻击 |
| 💀 **覆灭机制** | 当队伍所有成员进入旁观者模式，该阵营生物将不再生成 |

### 队伍机制

```
┌─────────────────────────────────────────────────────┐
│                    队伍结构                          │
├─────────────────────────────────────────────────────┤
│  👑 队长                                             │
│    ├── 持有契约                                      │
│    ├── 决定阵营                                      │
│    └── 队伍编号                                      │
│                                                      │
│  👤 队员                                             │
│    ├── 共享队长契约                                  │
│    ├── 共享队伍颜色                                  │
│    └── 可传送到队友身边                              │
└─────────────────────────────────────────────────────┘
```

### 契约保护机制

- **同阵营免疫**：契约模组的生物不会攻击你
- **负面效果免疫**：契约模组的负面药水效果无效
- **队友保护**：无法攻击同队玩家
- **仇恨转移**：契约生物会自动攻击敌对阵营

### 📜 生灵契约

**核心道具 - 建立阵营连接**

```
┌─────────────────────────────────────────────────────────────┐
│                       📜 生灵契约                            │
├─────────────────────────────────────────────────────────────┤
│                                                              │
│  用法：                                                      │
│    • 右键生物 → 与该生物所属模组签订契约                     │
│    • Shift+右键 → 解除当前契约                              │
│                                                              │
│  效果：                                                      │
│    • 该模组的所有生物不再攻击你                              │
│    • 该模组的负面效果对你无效                                │
│    • 该模组生物会攻击敌对阵营                                │
│                                                              │
│  注意：                                                      │
│    • 队员自动共享队长的契约                                  │
│    • 解除个人契约后，若在队中仍共享队长契约                  │
│                                                              │
└─────────────────────────────────────────────────────────────┘
```

```
  ✓ 不能收服玩家
  ✓ 存储槽位未满

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
```

### 随从行为

| 行为模式 | 描述 |
|:---:|:---|
| 🚶 跟随 | 跟随主人移动，保持适当距离 |
| 🛡️ 防御 | 攻击靠近主人的敌人 |
| ⚔️ 攻击 | 主动寻找并攻击敌人 |

### 随从特性

- **不会攻击主人**：即使误伤也不会造成伤害
- **不会消失**：持久化存在
- **自动攻击**：攻击主人攻击的目标
- **保护主人**：攻击攻击主人的敌人
- **驯兽师加成**：如果主人是驯兽师，随从获得额外属性

```
  • 目标必须是同队玩家
  • 不能传送到自己

效果：
  • 瞬间传送到目标玩家位置
  • 目标玩家会收到通知

示例：
  /contract team tp Steve

示例输出：
  §a[生灵契约] §f已传送到队友 §eSteve §f身边。
  （目标玩家收到）§a[生灵契约] §f队友 §ePlayer §f传送到了你身边。

错误提示：
  §c你不能传送到你自己！
  §c只能传送到同一队伍的玩家！
```
